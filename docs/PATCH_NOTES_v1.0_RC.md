# CardVille v1.0-rc.1 Patch Notes

## 목표

v1.0-preview에서 발견될 수 있는 디자인 촌스러움, 메인 로비 과밀, 카드/에셋 활용 부족, 대형 에셋팩 성능 위험을 줄이고 출시 후보급 구조로 다듬는 패치입니다.

## 적용 내용

- 메인 로비 전체 재배치
- 월드 배경 선택 화면 추가
- 카드 뒷면 선택 화면 추가
- 카드팩 확률표 추가
- 카드 상세 보기 화면 추가
- 컬렉션 카드 터치 상세 보기 연결
- 보상 화면 카드팩 PNG 사용
- 선택 월드 배경을 로비/서고/플레이/보상/앨범 화면에 적용
- ThemeSystem 추가
- UI 좌표 검사 도구 추가
- verify 파이프라인에 check:ui 추가

## 검증

```bash
npm run verify
```
