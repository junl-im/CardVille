# CardVille 1.0.7 Patch Notes

## 목적
1.0.6에서 실행은 되었지만 일부 환경에서 글자가 배경과 섞이거나 너무 약하게 보이는 문제를 수정했다.
이번 패치는 기능 추가가 아니라 UI/UX 가독성, 텍스트 대비, 패널 대비, 버튼 터치 영역을 우선 안정화한다.

## 적용 내용
- 전역 한국어 폰트 스택 추가
- Phaser Text에 stroke/shadow/resolution 기본 스타일 추가
- 제목/본문/보조문구/카드 글자 스타일 분리
- 글래스 패널을 더 어둡고 선명한 대비로 조정
- 배경 밝기와 패널 대비 조정
- 모든 주요 Scene 텍스트 스타일 재적용
- 버튼 라벨 가독성 강화
- 버튼 터치 hitbox 확대
- 카드 앞면 텍스트 대비 강화
- 버전 1.0.7 표기

## 검증
- npm run build 통과
- npm run verify 통과
