# CardVille 1.0.8 Patch Notes

## 목적
1.0.7 실행 후 확인된 버튼 터치 매핑 불안정, 화면 문구 잔여 세계관 문제, CardVille 브랜드 정리를 수정한 안정화 패치입니다.

## 수정 사항
- `꿈의 서고` 잔여 문구 전부 제거
- 메인 흐름을 `카드마을 광장 → 게임 선택 → 스테이지 선택 → 플레이`로 정리
- `낱말의 책`, `연산의 책` 등 책 콘셉트 문구를 `낱말 카드`, `연산 카드`로 변경
- `GameButton`을 컨테이너 hitArea 방식에서 내부 `Zone` 방식으로 변경
- 버튼의 보이는 영역보다 실제 터치 영역을 더 크게 확장
- `onClick()` API 추가로 모바일 터치 이벤트 처리 흐름 단순화
- `?touchDebug` 쿼리로 버튼 터치 영역 시각화 가능
- StageSelect / ModeSelect 터치 영역 확대
- CardVille 마을형 배경으로 정리

## 검증
- `npm run build` 통과
- `npm run verify` 통과
- 프로젝트 내부 `꿈의/서고/Dream` 사용자 노출 문구 제거 확인
