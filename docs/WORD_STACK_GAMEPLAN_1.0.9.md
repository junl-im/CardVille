# Word Stack Game Plan 1.0.9

CardVille의 핵심 플레이는 다음 구조로 정리합니다.

1. 상단 HUD: 코인, 레벨, 설정
2. 좌측 패널: 스텝, 힌트, 셔플
3. 상단 목표 카드: 현재 찾아야 하는 말 카드 계열
4. 중앙 보드: 여러 컬럼에 세로로 쌓인 말 카드
5. 규칙: 각 컬럼의 맨 위 카드만 선택 가능
6. 목표와 같은 계열을 고르면 카드가 제거되고 목표 진행도가 오른다
7. 모든 목표를 끝내면 보상 카드팩으로 이동한다

이번 버전은 시스템의 첫 번째 안정 프로토타입이며, 다음 버전부터 콤보, 잠금 해제, 광고 대체 보상, 카드팩 연출을 단계적으로 붙입니다.
