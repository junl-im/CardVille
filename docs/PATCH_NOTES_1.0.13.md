# CardVille 1.0.13 Patch Notes

## 목표

실행 안정성이 확보된 1.0.12를 기준으로, 모바일 세로 화면의 양옆 여백을 줄이고 게임 화면을 더 카드게임답게 꾸민다.

## 변경

- PlayScene 카드 보드 폭 확대
- 좌측 스텝/힌트/셔플/나가기 패널 슬림화
- 말 카드 크기 확대
- 목표 카드 영역 확대
- 기존 AquaGlass PNG 에셋 일부 재도입
- 카드마을 배경 이미지 적용
- HUD 코인/별/설정 아이콘 적용
- 게임관/앨범/카드팩 아이콘 적용
- 카드 뒷면 이미지 적용
- 보상/앨범 카드에 희귀도 프레임 일부 적용
- 1.0.13 버전 표기 정리

## 안전 정책

- SVG 사용 금지 유지
- Service Worker 캐시 비활성 유지
- 대형 카드 이미지 전체 로딩은 아직 보류
- 게스트 즉시 시작, 말 카드 스택 플레이, 결과/보상/앨범 흐름 유지
