# UI / Engine Upgrade 1.0.14

## 유지한 안정 정책
- Service Worker/PWA 캐시는 계속 비활성화.
- 5000장 카드 이미지는 강제 preload하지 않음.
- Firebase는 게스트 시작 흐름에 개입하지 않음.
- SVG는 사용하지 않음.

## 화면 비율 정책
- 기준 해상도: 390x844.
- 좌측 보조 패널은 58px 폭으로 축소.
- 보드 시작 x는 68px, 폭은 316px로 확대.
- 네 개 카드 컬럼은 x = 108, 187, 266, 345로 배치.
- 카드 크기는 76x96으로 확대.

## 다음 작업 후보
- 실제 기기에서 카드 아래쪽 겹침 여부를 확인한 뒤 gapY를 세부 조정한다.
- 카카오 브라우저에서 pointerup/zone 충돌을 다시 점검한다.
- 큰 에셋은 씬별 lazy load 방식으로만 연결한다.
