# CardVille 1.0.9 Patch Notes

## 목적

실행 안정화 이후, 사용자가 제시한 말 카드형 모바일 게임 구성을 기준으로 플레이 구조를 다시 정리했습니다.

## 변경 사항

- 플레이 화면을 말 카드 스택형 보드로 교체
- 상단 코인/레벨 HUD 추가
- 좌측 스텝/준비중 보조 패널 추가
- 목표 카드와 같은 계열의 말 카드를 고르는 규칙 추가
- 카드가 세로로 쌓이는 컬럼 구조 적용
- 버튼 hitbox 과확장 제거
- 실제 버튼과 터치 영역의 위치를 일치시키는 방식으로 수정
- 게임 선택을 말 카드 중심으로 정리
- 카드마을 광장, 게임관, 앨범 흐름 유지

## 검증

- `npm run build`
- `npm run verify`
- `npm run check:brand`
