# Comfort UI Audit 1.0.15

## 발견한 문제

- 첫 로그인 화면 하단 패널이 너무 어둡고 커서 키아트가 덜 살아났다.
- 기존 버튼 PNG에는 자체 글자가 들어 있어 Phaser 텍스트와 겹칠 수 있었다.
- 카드 프레임과 shine 효과가 카드 본문 위에 얹히면서 일부 화면에서 투명하거나 조잡해 보일 수 있었다.
- 보드 레일과 패널의 알파값이 낮아 배경이 비쳐 피로감을 줄 수 있었다.

## 조치

- 로그인 화면은 키아트 전체를 유지하고 하단에 작은 컨트롤 영역만 둔다.
- plain 버튼 PNG 6종을 추가하고 GameButton이 이 스킨을 사용한다.
- 카드 프레임은 카드 뒤쪽 장식으로 이동하고 카드 본문은 완전 불투명 크림색으로 고정한다.
- HUD, 사이드 패널, 목표 영역, 보드 레일의 불투명도를 높인다.
- 배경은 색감을 유지하되 카드 뒤쪽 노이즈를 줄이는 comfort overlay를 적용한다.
