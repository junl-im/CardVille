# UI/UX & Content Audit 1.0.11

## 체크한 문제

- 모바일 세로 화면에서 버튼과 카드가 겹치는지
- 카드 보드가 좌측 스텝 패널과 충돌하는지
- 말 카드 글자가 카드 안에서 읽히는지
- 스테이지가 4개만 있어 콘텐츠가 짧은지
- 카드 앨범이 12개 이상에서 정리되는지
- 보상 카드가 단순 랜덤 1개라 반복감이 큰지
- SVG 금지 정책이 자동 검사되는지

## 개선

- 스테이지 선택은 페이지 단위로 분리
- 카드 앨범은 페이지 단위로 분리
- 플레이 보드에 레일 배경을 추가해 중앙 영역을 명확하게 표시
- 카드 텍스트 크기는 글자 수에 따라 자동 축소
- 보상 카드 희귀도와 카드풀 확장
- UI 검사 스크립트로 필수 시스템 파일과 SVG 존재 여부 확인

## 아직 남은 과제

- 실제 휴대폰 화면에서 카드 하단/상단 여백 추가 확인
- 카카오 브라우저에서 터치 좌표 재확인
- 카드 제거 후 더 부드러운 낙하/재정렬 애니메이션 추가
- 카드팩 오픈 연출 고급화
- 실제 PNG/WebP 카드 아트 재도입
