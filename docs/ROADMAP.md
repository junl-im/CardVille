# CardVille Roadmap

## v0.1

- Phaser 3 + TypeScript + Vite 스타터
- Firebase 연결
- GitHub Pages 자동 배포
- 낱말 카드 프로토타입

## v0.2

- Splash → Loading → Login → Main Lobby → Mode Select → Stage Select → Play → Result → Reward 흐름 정리
- 에셋 폴더 구조 생성
- asset manifest 추가
- 낱말 카드 3스테이지 확장
- 그림 + 텍스트 카드 데이터 구조 추가
- 카드 등급 common, uncommon, rare, epic, legendary, mythic 확장

- 컬렉션 저장 및 앨범 반영

## v0.4 예정
