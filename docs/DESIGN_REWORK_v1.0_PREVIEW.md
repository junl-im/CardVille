# CardVille v1.0-preview Design Rework

## 디자인 문제 진단

기존 디자인은 기능 구조는 잡혀 있었지만, 다음 문제가 있었습니다.

- 이모지 의존도가 높아 임시 프로토타입 느낌이 강함
- 배경이 절차형 그래픽 중심이라 세계관 밀도가 낮음
- 버튼과 패널이 반복되어 장면별 차별감이 부족함
- 대형 에셋팩이 실제 게임 UI와 연결되지 않음

## 개선 방향

v1.0-preview에서는 실제 PNG 에셋을 연결해서 다음 방향으로 개선했습니다.

- 꿈의 서고 배경을 실제 래스터 이미지로 표시
- 카드 프레임 PNG를 카드 컴포넌트에 연결
- 배경은 `VisualSystem`에서 통합 관리
- 5,000장 카드 이미지는 별도 도감에서 확인 가능
- 큰 에셋은 lazy-load 방식으로 성능 보호

## 다음 디자인 목표

v1.0-preview 이후에는 다음이 필요합니다.

1. 책 선택 화면을 실제 책 오브젝트 이미지 기반으로 변경
2. 카드팩 오픈 화면을 실제 팩 이미지 기반 애니메이션으로 변경
3. 카드 상세 화면 추가
4. 월드 배경 선택 시스템 추가
5. 카드 프레임/뒷면/광택/그림자를 완전한 조립식 카드 렌더러로 분리
