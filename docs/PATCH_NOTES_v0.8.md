# CardVille v0.8.0 Patch Notes

## 목표

v0.8.0은 기능 추가보다 실제 서비스 품질을 높이는 데 초점을 맞춘 대규모 패치입니다.
UI 겹침, 카드 배치, 카드 연출, 컬렉션 페이지 넘침, 성능 예산, 데이터 검증, 에셋 정책을 함께 정리했습니다.

## 적용 내용

- 퍼즐 카드 모드 오픈
- 4개 모드 12스테이지에서 5개 모드 15스테이지로 확장
- 컬렉션 24종에서 48종으로 확장
- 컬렉션 화면 페이지네이션 추가
- 컬렉션 필터 추가
- 카드 등장 애니메이션 추가
- 카드 선택 시 부유감/광택 개선
- 오답 흔들림 후 위치 복구 안정화
- PlayScene 카드 배치 동적 계산
- 결과 화면에 이동 수, 실수 수, 플레이 시간 표시
- PerformanceSystem 추가
- LayoutSystem 추가
- VisualSystem 추가
- 설정의 절전/데이터 절약 옵션이 파티클과 앰비언트 오브젝트 수에 반영
- SVG 금지 검사 추가
- 번들 용량 예산 검사 추가
- service worker 캐시 v0.8 갱신

## 안정성 개선

- `npm run verify`에 asset policy와 bundle budget 검사를 추가했습니다.
- 퍼즐 모드 JSON이 catalog와 service worker에 함께 등록됩니다.
- JSON 검사에서 카드팩 확률, 슬롯 수, 보장 등급, 세트 참조를 검증합니다.

## 주의

아직 실제 WebP 카드 이미지는 포함하지 않습니다.
현재는 이모지 기반 프로토타입 렌더링이며, 실제 WebP/PNG 에셋이 들어오면 imageKey 기준으로 연결하면 됩니다.
