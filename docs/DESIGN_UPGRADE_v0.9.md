# CardVille v0.9 Design Upgrade

## 개선 방향

촌스러운 교육 앱 느낌을 줄이기 위해 다음 원칙으로 정리했다.

- 단색 배경 대신 레이어드 글래스 배경
- 버튼에 광택, 그림자, 상단 하이라이트 추가
- 패널에 은은한 유리 질감과 내부 반사광 추가
- 메인 로비에서 이모지 의존도를 낮추고 Dream Library 브랜드 강조
- 카드에 실제 WebP 그림 표시
- 과한 장식보다 고급스러운 색감과 여백 확보

## 아직 남은 과제

- 최종 AI 원화 교체
- 카드팩 전용 오픈 애니메이션 고급화
- 책 선택 화면 3D 책 오브젝트화
- 월드별 배경 실사급 스타일 통일
- 사운드/햅틱 디테일 추가
