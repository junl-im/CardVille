# Design Pass v1.0-rc.1

## 개선한 문제

1. 메인 로비가 버튼 나열처럼 보여 브랜드감이 약함
2. 선택 월드와 카드 뒷면이 사용자 경험에 반영되지 않음
3. 카드팩 확률/카드 상세 보기처럼 수집형 게임에 필요한 정보 화면 부족
4. 대형 에셋이 있음에도 실제 UI에 반영되는 지점이 부족함
5. 작은 화면에서 버튼과 하단 표기가 겹칠 가능성 존재

## 적용 방향

- 로비를 히어로 패널 + 2x2 기능 타일 + 시스템 도크로 재구성
- 월드 선택과 카드 뒷면 선택을 별도 화면으로 분리
- 카드 상세 보기로 수집 보상 가치를 강화
- 카드팩 확률표로 보상 투명성 강화
- 선택한 월드를 주요 화면 배경으로 반영

## 남은 디자인 과제

- 실제 사운드와 햅틱 타이밍 정교화
- 책 오브젝트 자체를 PNG/Sprite 기반으로 교체
- 카드팩 오픈 프레임 애니메이션 고도화
- 컬렉션 세트 완성 보상 팝업 추가
