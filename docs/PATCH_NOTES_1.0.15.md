# CardVille 1.0.15 Patch Notes

## 핵심

- 첫 로그인 화면을 더 꽉 찬 풀스크린 플레이 화면처럼 재구성했다.
- 로그인 하단 컨트롤을 키아트를 가리지 않는 위치로 재배치했다.
- 카드 앞면, 목표 카드, 보드 레일의 과한 투명감을 제거했다.
- 버튼 PNG를 텍스트 없는 plain 스킨으로 교체해 버튼 글자 겹침을 줄였다.
- 패널과 HUD는 더 어둡고 단단한 톤으로 조정해 눈이 편하도록 했다.

## 유지

- 게스트 즉시 시작
- 인트로 영상
- 휴대폰 뒤로가기 팝업
- 말 카드 스택 플레이
- 힌트/셔플/콤보/보너스
- 카드팩 오픈 보상
- SVG 금지
- Service Worker 비활성

## 검증

`npm run verify` 기준으로 빌드, 배포 파일, 브랜드, UI/콘텐츠, 레이아웃/에셋 검사를 수행한다.
