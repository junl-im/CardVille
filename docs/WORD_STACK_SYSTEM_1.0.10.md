# Word Stack System 1.0.10

CardVille의 기본 플레이는 말 카드 스택 정리 방식이다.

## 기본 규칙
1. 화면 상단 목표 카드의 계열을 확인한다.
2. 각 컬럼의 맨 위 카드 중 목표 계열과 맞는 카드를 선택한다.
3. 정답이면 카드가 제거되고 점수와 콤보가 오른다.
4. 오답이면 스텝이 줄고 콤보가 초기화된다.
5. 목표 수량을 채우면 다음 목표로 넘어간다.
6. 모든 목표를 완료하면 결과와 보상 화면으로 이동한다.

## 보조 기능
- 힌트: 현재 맨 위 카드 중 정답 가능한 컬럼을 강조한다.
- 셔플: 남은 카드를 다시 섞는다. 대신 스텝 1을 사용한다.

## 보상
- 별 개수, 점수, 최고 콤보에 따라 XP와 코인 보상이 늘어난다.
- 보상 카드팩은 카드 앨범에 저장된다.
