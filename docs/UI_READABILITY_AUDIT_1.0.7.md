# UI Readability Audit 1.0.7

## 발견된 문제
- 흰색 텍스트가 밝은 Aqua Glass 배경과 섞일 수 있음
- 일부 작은 텍스트가 모바일에서 흐리거나 얇게 보일 수 있음
- 카드/보상/앨범 화면의 작은 라벨 대비가 약함
- 버튼 시각 영역과 실제 터치 영역이 충분히 크게 보장되어야 함

## 수정 원칙
- 모든 밝은 텍스트에 어두운 stroke를 적용한다.
- 모든 어두운 텍스트에 흰색 stroke 또는 밝은 그림자를 적용한다.
- 패널은 투명하되 실제 텍스트 뒤에는 어두운 면을 확보한다.
- 버튼은 보이는 크기보다 큰 hitbox를 가진다.
- 한국어 렌더링을 위해 Noto Sans KR / Apple SD Gothic Neo / Malgun Gothic 폰트 스택을 사용한다.
