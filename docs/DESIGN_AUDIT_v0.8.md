# CardVille v0.8 Design / UI Audit

## 발견한 문제

1. 컬렉션 24장 이상에서 카드가 화면 아래로 내려가는 문제가 있었습니다.
2. 카드가 정답 처리될 때 즉시 사라져 보상감이 약했습니다.
3. PlayScene 카드 좌표가 고정 배열이라 카드 수가 바뀌면 배치가 깨질 수 있었습니다.
4. 파티클과 별빛이 많은 화면에서 저사양 모바일 부담이 커질 수 있었습니다.
5. 결과 화면에 점수 외 플레이 품질 지표가 부족했습니다.
6. 검증 스크립트가 JSON 구조 전체를 충분히 점검하지 못했습니다.

## 수정 방향

- 컬렉션은 12장 단위 페이지네이션으로 전환했습니다.
- 카드 선택/정답/오답 애니메이션을 안정화했습니다.
- LayoutSystem으로 카드 배치를 계산하게 했습니다.
- PerformanceSystem으로 절전/데이터 절약 옵션을 반영합니다.
- 결과 화면에 이동, 실수, 시간 정보를 추가했습니다.
- SVG 금지, 번들 예산, 카드팩 데이터 검사를 추가했습니다.

## 디자인 기준

- 배경은 어두운 꿈의 서고 톤을 유지합니다.
- 카드와 버튼은 밝은 Aqua Glass 대비를 유지합니다.
- 고급스러운 느낌을 위해 반사광과 그림자는 절제해서 사용합니다.
- 모바일 한 손 조작을 위해 핵심 버튼은 하단 중앙에 배치합니다.
